Simple Banner AST Tag

Simple SafeFrame Banner AST Tag

Instructions

Below you can see the code used to define the 2 tags implemented on this page through AST:

apntag.defineTag({
tagId: 12886199,
sizes: [[300,250]],
targetId: 'apn_ad_slot_1'
});

apntag.defineTag({
enableSafeFrame: true,
tagId: 12992714,
sizes: [[300,250],[738,90]],
targetId: 'apn_ad_slot_2'
});

It's important to note that before defining the tags, two other steps need to be implemented in order for this to work. Please follow the implementation documentation which can be found here: AppNexus Wiki - Set up Placements with AST.

For further AST API references, you can refer to the following documentation: AppNexus Wiki - AST API Reference.

Further things to note:

  • Setting allowedFormats on AST is optional; if it's set, it will override the media types assigned on the placements' settings in Console.
  • The media types allowed on the allowedFormats field are banner, video, native and audio. If you want a different format than these on the placements, then don't set the field and set the media type on the placement in Console.
  • It is possible to have several formats on the same placement (superauction).
  • As it can be seen in the above code, placement 12992714 has enableSafeFrame: true. This means it enables the SafeFrame iFrame. More information on this can be found here: AppNexus Wiki - SafeFrame API Reference.

With the help of AST, roadblock can also be achieved. Please see this page for the setup: Demo Page- Banner Roadblock AST Implementation.
